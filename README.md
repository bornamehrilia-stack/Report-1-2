### ۱. هدف آزمایش

هدف از این آزمایش، یادگیری نحوه مدیریت چندین پین خروجی به صورت هم‌زمان و ایجاد تاخیر زمانی متناسب برای کنترل توالی روشن و خاموش شدن دو قطعه (LEDها) با استفاده از برد آردوینو است.

### ۲. شرح مختصر مدار

در این پروژه، دو عدد LEDبه ترتیب به پین‌های شماره **۸** و **۹** برد **Arduino Uno** متصل شده‌اند. منطق برنامه به گونه‌ای است که در هر لحظه یکی از LEDها روشن و دیگری خاموش باشد. این عمل با یک وقفه یک ثانیه‌ای تکرار می‌شود تا وضعیت الاکلنگی در خروجی مشاهده شود.

### ۳. موارد مورد نیاز

* ۱ عدد برد **Arduino Uno**
* ۲ عدد **LED**
* ۲ عدد **مقاومت** (جهت حفاظت و محدود کردن جریان LEDها)
* برد بورد (Breadboard)
* سیم‌های رابط (Jumper Wires)

### ۴. نمایش کد پروژه

```cpp
void setup() {
  // تنظیم پین‌های 8 و 9 به عنوان خروجی دیجیتال
  pinMode(8, OUTPUT);
  pinMode(9, OUTPUT);
}

void loop() {
  // وضعیت اول: سبز روشن، قرمز خاموش
  digitalWrite(9, HIGH); 
  digitalWrite(8, LOW); 
  delay(1000); // توقف به مدت 1 ثانیه

  // وضعیت دوم: قرمز روشن، سبز خاموش
  digitalWrite(8, HIGH); 
  digitalWrite(9, LOW); 
  delay(1000); // توقف به مدت 1 ثانیه
}

```

### ۵. توضیح مختصر کد

* **بخش `setup`:** در این بخش پین‌های ۸ و ۹ که به LEDها متصل هستند، به عنوان خروجی (`OUTPUT`) پیکربندی شده‌اند تا بتوانیم به آن‌ها ولتاژ بفرستیم.
* **بخش `loop`:** * ابتدا ولتاژ پین ۹ را بالا (`HIGH`) و پین ۸ را پایین (`LOW`) می‌بریم.
* سپس ۱۰۰۰ میلی‌ثانیه (۱ ثانیه) صبر می‌کنیم.
* در مرحله بعد، وضعیت را برعکس می‌کنیم تا LED دیگر روشن شود.
* این چرخه باعث می‌شود LEDها به صورت متناوب و با فواصل زمانی منظم چشمک بزنند.
